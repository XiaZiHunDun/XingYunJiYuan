# 专家方案设计 · 邀请提示词（可直接复制使用）

> 用途：邀请出版/网文编辑、内容策划、IP 评估或资深创作者，为《星陨纪元》的**修订与质量提升**提供可落地方案。  
> 使用方式：复制下方「---」之间的正文，按需删改括号内可选项后发给对方。

---

## 角色与目标

你是一位在**长篇小说策划、修订流程、网文/出版双赛道**至少一个领域有成熟经验的专家。请基于我方提供的背景与约束，设计一套**可执行、可验收**的《星陨纪元》**评审—优化—交付**总体方案（而非单章润色）。

**你的交付目标**：让我方团队能按你的方案分工、排期、质检，并在 3–6 个月内看到**可度量**的进展（具体周期可由你根据方案推算）。

---

## 一、作品与项目背景（事实陈述）

1. **作品**：《星陨纪元》，类型为**玄幻修真 + 星际科幻**，三卷共 **360 章**（第一卷 ch001–ch120，第二卷 ch121–ch240，第三卷 ch241–ch360）。正文为 Markdown 分章存储，另有全卷合并稿便于通读。

2. **已知结构性问题**（基于统计与通读）：
   - 第一卷单章篇幅明显长于第二、三卷；第二、三卷在批量重写阶段被**过度压缩**，存在节奏仓促、细节与人物厚度不足的风险。
   - 部分卷间衔接、时间跳跃、个别过短章节已被标注为风险点（例如第三卷个别章节极短、千年跳跃是否突兀等）。

3. **我方已整理的内部资产**（你可假设存在且会同步给你查阅）：
   - **评审指令**：`review_prompt.md`（含主角唯一性、修炼里程碑、卷间衔接检查、评审报告模板）。
   - **方法论 v0.4**：Hard/Soft 分层、Gate 顺序、扩写基线不退、软指标量表、扩写与重写边界、篇幅主指标（§4）、**§九 评审作业规范**（人/机分工、读窗、缺陷分级、抽样、记录模板）。
   - **操作手册**：章节扩展 SOP（扩写前对表、扩写中反注水、扩写后门禁）。
   - **设定追踪**：境界进度、角色状态、伏笔追踪及项目计划类文档。

## 二、不可协商的创作约束（请在你的方案中视为硬边界）

以下约束来自我方已定稿的 IP 与叙事策略，**方案不得建议违背**；你可以在「如何检查、如何防止回归」上加强：

1. **叙事主角唯一性**：全书以 **林夜 + 苏琳** 为唯一主线主角；不得出现「换主角 / 接班人叙事夺权 / 代际传承式 handoff」等处理。
2. **「传承」语义边界**：禁止的是**叙事主线交给新一代**；世界观内的功法、记忆、物品等设定传承若不与换主角绑定，可按个案保留，但需在质检中区分。
3. **修炼里程碑与卷间关键衔接**：须与现有 `review_prompt.md` 及境界追踪表一致；方案若涉及改剧情，须单独标注**对里程碑的影响**与迁移成本。
4. **质量门禁**：任何扩写/重写批次必须通过 **Hard 门禁**（对照评审指令），并对 **Soft 指标（剧情完整性、逻辑、文笔、情感、节奏）** 有最低标准。

## 三、请你回答的核心问题（方案必须覆盖）

请按优先级组织你的回答，并显式说明**取舍理由**：

**A. 战略取舍**

1. 在「三卷阅读体验统一」「保留第一卷现有密度」「控制总工期」三者中，你建议的**优先级排序**是什么？若只能保两项，你砍哪一项、如何补偿读者体验？
2. 你更推荐以哪种**北极星指标**驱动迭代：（a）单章有效字数区间；（b）卷内节拍/冲突密度；（c）先清零 Hard 风险再谈体验统一？是否建议混合指标？权重如何？

**B. 战术路径**

3. **第二、三卷**：以「分阶段扩展」为主、还是「局部重写」为主、或「混合」？请给出**判定标准**（何种情况扩写、何种情况必须重写）。
4. **第一卷**：在整体方案中，对「偏长章」的处理策略是什么（维持 / 仅删水 / 与二三卷对齐压缩）？请给出**触发条件**（例如字数阈值、重复信息判定方式）。

**C. 流程与组织**

5. 建议的**角色分工**（主编、设定监理、扩写执行、统稿、终稿评审）最小配置是什么？哪些环节适合人机协同、哪些必须人审？
6. 请给出**分阶段路线图**（例如：基线评审 → P0 修复 → 扩写试点 → 规模化 → 全书统稿），每阶段的**输入、输出、验收标准、预估人天量级**（可给区间）。
7. 如何设计**版本与回归测试**，避免已修复问题（如特定章节禁忌表述、境界描述）在后续扩写中复发？

**D. 风险与商业（可选但欢迎）**

8. 若目标读者为网文平台连载读者 vs 出版单行本读者，你的方案在**节奏与章长**上分别如何调整？若我方暂不定受众，你建议的**默认假设**是什么？
9. 你认为本方案最大的**三个风险**与对应的**缓释措施**是什么？

## 五、行为约束

- **基于给定事实与约束推理**；若信息不足，请在第 8 节列出需补充材料，**不要虚构**我方未提供的设定或数据。
