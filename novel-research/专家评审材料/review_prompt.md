# 《星陨纪元》小说评审提示词

## 作品基本信息

**作品名称**: 星陨纪元
**类型**: 玄幻修真+星际科幻
**结构**: 三卷本，共360章
- 第一卷（ch1-120）：废土星火 - 林夜废土崛起，与苏琳相识
- 第二卷（ch121-240）：星际争锋 - 星辰炼狱试炼，突破创世境，打败暗皇
- 第三卷（ch241-360）：永恒守护 - 守护宇宙，对抗虚无之主

**核心设定**:
- **唯一主角**: 林夜 + 苏琳（贯穿全书1-360章，是唯一主角）
- **修炼体系**: 凡人之躯 → 星尘境 → 粒子境 → 星火境 → 脉冲境 → 裂变境 → 黑洞境 → 创世境
- **世界观**: 宇宙中存在光明与黑暗的平衡，暗域是威胁，星辰宗是上古最强门派

## 评审要求

### 一、主角唯一性检查（最重要）

请特别关注以下几点：

1. **是否始终以林夜+苏琳为核心？**
   - 全文不能出现林夜苏琳之外的角色作为主要视角
   - 不能有"传承给新一代"的情节
   - 不能有"继承者""接班人"等设定

2. **配角定位检查**
   - 铁蛋、莫言、小九等是配角，不能成为主线主角
   - 周明辉、林风等是辅助角色
   - 所有角色不能喧宾夺主

3. **常见问题排查**
   - 是否有"XX继承遗志""XX传承衣钵"等情节？
   - 是否有新的主角或主角团出现？
   - 是否有代际传承的暗示？

### 二、修炼体系一致性检查

1. **境界进展是否合理？**
   - ch121-130: 星辰炼狱 → 黑洞境
   - ch144: 星辰之心共鸣（暂时创世境级别力量）
   - ch147: 实际境界为黑洞境巅峰
   - ch160: 融合黑暗本源 → 创世境
   - ch195: 宇宙意识确认已达创世境

2. **战斗力是否自洽？**
   - 黑洞境 vs 创世境 的差距是否体现？
   - 敌人实力与主角实力的对比是否合理？

### 三、剧情衔接检查

1. **卷内衔接**
   - 章节之间是否连贯？
   - 是否有突兀的时间跳跃或情节断层？

2. **卷间衔接**
   - ch120 → ch121（进入星辰炼狱）
   - ch145 → ch146（打败暗皇后）
   - ch160 → ch161（突破创世境）
   - ch195 → ch200（大灾变后）
   - ch240 → ch241（虚无之主威胁）

3. **伏笔与回收**
   - 前面埋下的伏笔后面是否回收？
   - 是否有前后矛盾的剧情？

### 四、时间线检查

1. **是否清晰？**
   - 第一卷：废土时期（约1年）
   - 第二卷：星际争锋（30-50年）
   - 第三卷：永恒守护（数千年）

2. **时间跳跃是否合理？**
   - ch244: 千年时间跳跃是否突兀？
   - ch239-240: 第二卷结尾的时间点

### 五、世界观一致性检查

1. **势力设定**
   - 玄机阁、万剑宗、星辰会等势力
   - 暗域、暗皇的设定

2. **关键物品/地点**
   - 星辰之心、星辰炼狱、枢纽城
   - 星辰殿、暗渊之心、虚无之境

### 六、文笔与风格检查

1. **语言风格是否统一？**
2. **描写是否精彩？**
3. **对话是否自然？**

### 七、可读性/吸引力检查（S6）

请特别关注本章是否具备以下要素：

1. **钩子密度**
   - 开头200字内是否有冲突或悬念？
   - 结尾是否留下未解之谜或期待感？
   - 章中是否有小转折或信息释放？

2. **情绪高点**（满足至少1项为合格）
   - 爽点：逆转、打脸、实力压制、羁绊被认可
   - 泪点：牺牲、离别、误解后和解
   - 笑点：意外反转、人物反差萌

3. **代入感**
   - 主角决策时是否写清代价？
   - 是否有清晰的恐惧与欲望？

4. **情感曲线完整**（最低标准：开头有情绪、发展有起伏、收尾有余韵）
   - 开头（0-200字）：什么情绪？（悬念/压力/好奇/紧张）
   - 发展（200-2000字）：情绪如何变化？（期待↑/紧张↑/疑惑）
   - 高潮（收尾前200字）：情绪高点是什么？（爽/哭/震惊/感动）
   - 结尾：是否留钩子让读者点下一章？

5. **记忆点检查**（读者会记住什么？）
   - 本章结束后，读者会记住什么？（一句话总结）
   - 这个记忆点是否有爆点台词/震撼场面/情感共鸣？
   - 还是只是一个「发生过的事情」？（如果是，本章需要修改）

6. **伏笔/钩子检查**
   - 伏笔：是否为后续情节埋线索？设置-回收间隔是否合理（5-30章）？
   - 钩子：章末是否留悬念让读者想点下一章？

**如果缺少任何一个要素，本章是「无效章节」，需要修改。**
