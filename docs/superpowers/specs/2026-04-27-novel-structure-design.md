## 三、反派动机强化

### 3.1 暗皇动机修改

**核心设定**：暗皇曾是远古星辰宗的守护者，因大灾变失去一切后走向极端

**全篇碎片化揭示**：
- ch049：星陨宗主提及"另一个候选人"走错了路
- ch076-079：战斗中插入暗皇起源闪回
- ch086：暗皇自述"这个宇宙从一开始就是错误的"
- ch161：决战时揭示"我的目标是重启，不是毁灭"
- ch162：林夜与暗皇的对话，理解而非单纯击败

**关键修改**：
```
ch161修改：在决战高潮处增加一段对话
暗皇："你以为我为什么要毁灭一切？因为创造本身就是错误。"
林夜："所以你想要回归虚无？"
暗皇："不。我想要的是重启。让一切回到原点，重新开始。"
```

### 3.2 本源之母动机修改

**核心设定**：宇宙意志的具象化，因孤独而矛盾

**揭示方式**：
- ch261-280：在"光明之源"章节中以对话形式揭示
- ch301-320：命运之源考验时展现她的孤独
- ch333：最终决战以"理解"结束，而非"打败"

**关键修改**：
```
ch333修改：最终对决不再是力量对决
本源之母："我存在了亿万年，没有一个生命能理解我。"
林夜："我理解。你的孤独，你的矛盾。"
本源之母："所以你愿意帮我？"
林夜："不。我帮你找到答案——不是毁灭，不是创造，是守护。"
```
