| 目录 | 说明 |
|------|------|
| `docs/专家设计/` | 外部/多专家给出的总体方案过程稿，供查阅取舍理由。 |

## 三、门禁与北极星（摘要）

1. **Gate 0 — Hard**：通过 / 不通过；不通过不得合入。  
2. **Gate 1 — 篇幅**：二三卷默认单章 **2000–3500 字**（统计口径见方法论 §4.4）；例外须主编特批并记 `HISTORY.md`。  
3. **Gate 2 — Soft**：S1–S5 量表；批次均分与「两项同时 ≤2」等规则见 **方法论 v0.4**；**扩写不得使 S1/S2 低于扩写前基线**。

详见：`novel-research/小说评审优化方法论_v0.4.md` §〇点五、§2.3、§2.4；**「怎么评」**见同文档 **第九章（评审作业规范）**。

## 四、阶段流程（摘要）

**基线评审 → P0 修复 → 扩写试点（POC）→ 规模化扩写/重写 → 统稿与终审。**

人天、角色 RACI、材料清单、决策记录表见 **`docs/修订路线图-定案.md`**。
