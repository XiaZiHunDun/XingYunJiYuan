## 核心问题

### 1. 代际传承叙事（违反宪法）
```
大纲：林夜苏琳为全书主角，贯穿1-360章
实际：ch201后林夜苏琳化为宇宙守护者淡出，
      ch202-240以铁蛋莫言为主角，讲述其成长、婚礼、传承
      铁蛋称呼"林夜叔叔""苏琳姐姐"，暗示代际传承
```

### 2. 主角更替（违反宪法）
```
大纲：全书唯一主角是林夜+苏琳
实际：ch201后林夜苏琳仅在ch204-205、ch227-240出场，
      其余25章以铁蛋莫言为主角
```

### 3. 关键情节缺失
```
大纲要求：
- ch201-205: 铁蛋&莫言婚礼（配角戏）
- ch221-240: 林夜&苏琳婚礼（主角戏）← 实际缺失

实际内容：
- ch215: 铁蛋&莫言婚礼（升格为主角戏）
- ch231: 铁蛋&莫言婚礼（重复）
- ch236-240: 林夜&苏琳婚礼（存在但顺序混乱）
```

## 修复要求

### 重写原则
1. **林夜苏琳必须始终是主视角**
   - 铁蛋莫言的戏份只能作为"副线"侧面描写
   - 不能有铁蛋主视角的独立章节

2. **时间线修正**
   - 移除"千年后"的超大时间跳跃
   - 恢复"50年"的合理时间跨度

3. **婚礼顺序**
   - ch201-205: 铁蛋&莫言婚礼（林夜苏琳旁观）
   - ch221-235: 林夜&苏琳婚礼（主角主场）
   - ch236-240: 击败暗皇后传承诺言

4. **禁止用语**
   - 删除"林夜叔叔""苏琳姐姐"等代际称呼
   - 改用"林夜""苏琳"或"老大""琳姐"
