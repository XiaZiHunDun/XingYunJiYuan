## 七、创作红线与评审指南

### 7.1 创作红线（宪法级·强制）

> **唯一核心规则**：全书主角为**林夜+苏琳**（贯穿1-360章）
> 以下红线任何时候都不得违背：

- ❌ 禁止代际传承叙事（林夜/苏琳不能把力量传给下一代）
- ❌ 禁止换主角叙事（不能以其他角色为主线视角）
- ❌ 禁止"传承衣钵""继承遗志"等情节
- ❌ 林月/星尘是辅助角色，不是主角

### 7.2 评审维度（指导级·参考）

| 维度 | 检查要点 | 说明 |
|------|----------|------|
| 修炼体系一致性 | ch121-130黑洞境、ch144星辰之心、ch160创世境 | 境界进展需合理 |
| 剧情衔接 | 卷内连贯性、卷间过渡、伏笔回收 | 避免断层 |
| 时间线 | 第一卷约1年、第二卷30-50年、第三卷数千年 | 跳跃需自然 |
| 世界观一致性 | 势力设定、关键物品/地点 | 保持统一 |
| 文笔与风格 | 语言统一、描写精彩、对话自然 | 质量达标 |
| 可读性/吸引力 | 钩子密度、情绪高点、情感曲线 | 读者留存 |
| 主角魅力 | 认同感、期待感、独特魅力点 | 角色塑造 |

### 7.3 篇幅要求

- 单章目标：**2000-3500字**（统计口径：正文汉字+全角标点）
- 当前平均：**3063字/章**
- 达标率：100%
