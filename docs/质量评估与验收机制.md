# 星陨纪元·质量评估与验收机制

> **创建日期**：2026-05-08
> **更新日期**：2026-05-08
> **用途**：定义项目"停止优化"的标准，防止无限迭代
> **类比**：大模型训练中的收敛机制（Loss Plateau、Early Stopping）
> **方法论来源**：`novel-research/小说评审优化方法论_v0.4.md` + `review_prompt.md`

---

## 一、核心原则

**项目收敛条件**：当继续优化的边际收益递减时，项目可以进入"可发布"状态。

**类比训练场景**：
- 模型训练：验证损失不再下降 → 停止训练
- 小说项目：H级问题为0 + S6均分≥3.5 + 连续2轮审查问题数无显著下降 → 停止优化

**优先级**：Hard(P0) > 篇幅(P1) > Soft(P2)

---

## 二、问题分级定义（与小说方法论对齐）

### H级 / Hard（必须修复，一票否决）

| 类型 | 定义 | 示例 |
|:---|:---|:---|
| 结构性错误 | 章节逻辑断裂、关键情节矛盾 | 主角被换、苏琳前后状态矛盾 |
| 宪法级违规 | 违反创作红线 | 代际传承叙事、换主角 |
| 章节缺失 | 关键章节文件不存在 | ch289等文件缺失 |
| 修炼体系违规 | 境界进展与里程碑不符 | ch160未达创世境 |

**处置**：禁止合入，必须回滚或重写

### S级 / Soft阻断（影响进入下一批次）

| 类型 | 定义 | 示例 |
|:---|:---|:---|
| S1 剧情完整性 | 章内目标-障碍-结果不清楚；卷内因果断档 | - |
| S2 逻辑自洽 | 时间线、战力、势力行为不合理 | - |

**S1/S2 ≤2分**：禁止进入下一扩写批次

### M级 / 中优（可合入，润色backlog）

| 类型 | 定义 | 示例 |
|:---|:---|:---|
| S3 文笔与风格 | 语言/对话/描写不统一 | 角色声音模板化 |
| S4 情感共鸣 | 无情绪波动 | 读者无法代入 |
| S5 节奏 | 拖沓/仓促/失衡 | 连续5章无起伏 |

**S3-S5任一为2，或均为3**：进入润色backlog

### L级 / 低优（通过）

| 类型 | 定义 |
|:---|:---|
| 全Soft ≥4 且无Hard疑点 | 通过，记录优秀样章 |

## 三、新增评分维度：S6/S7

### S6 可读性/吸引力（新增）

**核心问题**：读者是否愿意一口气读完？是否期待下一章？

| 分档 | 含义 | 表现 |
|:---|:---|:---|
| 5 | 钩子极强 | 爽点/笑点/泪点≥2处；强烈期待下一章 |
| 4 | 有明确钩子 | 情绪起伏良好；大部分读者会继续读 |
| 3 | 有基本吸引力 | 不突出；读者可能跳章 |
| 2 | 平淡乏味 | 缺乏钩子；读者可能弃文 |
| 1 | 完全无聊 | 流水账；读者会弃文 |

**简单记忆**："你想继续读吗？"

### S7 主角魅力/认同感（新增）

**核心问题**：读者是否认同这个主角？是否想看他成功？

| 分档 | 含义 | 表现 |
|:---|:---|:---|
| 5 | 主角魅力极强 | 读者强烈认同，渴望看到主角成功 |
| 4 | 主角有魅力 | 读者认同，愿意看主角故事 |
| 3 | 基本合格 | 缺乏独特魅力点 |
| 2 | 难以认同 | 主角行为让读者反感 |
| 1 | 完全无魅力 | 主角让读者反感 |

**简单记忆**："你认同这个主角吗？你想看他成功吗？"

## 七、收敛判断流程

### 7.1 审查阶段

执行审查时统计：
```
□ H级问题数量
□ 代际传承违规数量（grep -r "叔叔\|姐姐" src/）
□ S1-S7评分（S6/S7新增）
□ 篇幅达标率
□ 修炼体系矛盾
□ 文档与正文一致性
```

### 7.2 收敛条件

当**同时满足**以下条件时，项目进入"可发布"状态：

```
条件1：H级问题 = 0
条件2：代际传承违规 = 0
条件3：S6均分 ≥ 3.5
条件4：pronoun错误 < 10
条件5：连续2轮审查，问题数无显著下降
```

### 7.3 继续优化触发条件

- 发现新的H级问题
- S6均分 < 3.5
- 用户提出新的修改需求
- 文档与正文不一致
- 系统性错误（>3个同类问题）
