# 第五十二章 联盟之议

那天夜里，林夜在破晓号的会客厅里与星月长谈。

苏琳也在。

星月似乎早就料到了这个安排，特意为她留出了一个位置。铁蛋本想旁听，但被林夜用眼神制止了——有些事情，知道的人越少越好。

"说说你们的具体情况，"林夜开门见山，"三十七个人，能提供什么样的支持？"

星月点了点头，从袖中取出一块淡蓝色的晶石，放在桌上。

"这是星辰会的通讯结晶，"他说，"可以在一定范围内实现即时通讯，不受暗域的干扰。这是我们的技术之一。"

林夜拿起那块晶石，感受着其中蕴含的微弱能量波动。

"还有呢？"

"情报，"星月说，"星辰会在星际间有数百年的历史，积累了大量的情报网络。虽然近年来因为内部分裂损失了不少，但核心的情报渠道还在。我们可以提供暗域在各星域的部署情况、能量采集器的位置、以及——"

他停顿了一下。

"以及什么？"

"暗皇的近况，"星月的声音压低了一些，"我们有消息表明，暗皇正在筹划一次大规模的扩张行动。预计在三个月内，他将在多个星域同时发动进攻。"

林夜的眼神微微一凝。

三个月——这个时间比他想象的还要紧迫。

"具体是哪些星域？"

"破碎星域、玄机阁的核心领地、以及——"星月看向林夜，"废土。"

林夜的拳头不自觉地握紧了。

废土。

那是他的故乡，是林夜和苏琳从小长大的地方。如果暗域进攻废土，那里的人——那些没有反抗能力的普通人——将会遭受灭顶之灾。

"你们有办法阻止吗？"

"单独行动的话，很难，"星月坦诚地说，"但如果与我们联合，再加上玄机阁的支持，或许能够在局部地区形成有效的抵抗。"

"玄机阁会同意吗？"

"这正是我来找你的原因之一，"星月说，"玄机阁内部也有派系之争。一部分人主张与星辰会的温和派合作，共同对抗暗皇；另一部分人则认为应该置身事外、坐山观虎斗。"

"你希望我做什么？"

"我希望你能与玄机阁的高层会面，"星月说，"用你的影响力，推动他们站在正确的一边。"

林夜沉默了一会儿。

他不过是一个从废土里走出来的年轻人，从来没有与星际势力打交道的经验。让他去影响玄机阁的决策？

"我恐怕没有这么大的影响力。"

"你有，"星月认真地说，"你是星辰宗的唯一传人，是星陨圣殿试炼的胜利者，是——"

他停顿了一下。

"是什么？"

"是在废土上活下来的人，"星月说，"玄机阁的人敬重强者，尤其是从绝境中走出来的强者。你的经历本身，就是最好的证明。"

林夜没有说话。

他转头看向苏琳。

从始至终，苏琳一直没有开口。但她的眼神一直落在星月身上，带着某种审视的意味。

"苏琳，"林夜轻声问，"你怎么看？"

苏琳这才开口。

"我想先确认一件事，"她说，声音平静但带着几分冷淡，"你说你姐姐的死与星辰会内部有关——具体是谁？"

星月的表情变得有些痛苦。

"我还不能确定，"他说，"但我的怀疑对象是星辰会的副会长，沈渊。他一直是恐惧派的核心人物，与暗域有着说不清道不明的关系。"

"没有证据就不能乱说，"苏琳说，"我希望你明白这一点。"

星月点了点头。

"我知道，"他说，"所以我来找林夜，而不是直接在星辰会内部揭发他。在没有确凿证据之前，我不会轻举妄动。"

苏琳看了他一会儿，然后轻轻点了点头。

"这一点我可以接受，"她说，"但我还有一个问题。"

"请说。"

"你的联盟邀请，对象是林夜一个人，还是包括我们所有人？"

星月微微一愣，显然没想到她会问这个问题。

"当然是——"

"如果我们加入你的联盟，"苏琳打断了他，"铁蛋怎么办？莫言怎么办？他们也是林夜身边的人，也是从废土里走出来的人。他们有没有资格参与这个联盟？"

林夜看了苏琳一眼。

她的语气里带着几分尖锐，但他明白她想说什么。

星月的联盟邀请，从头到尾都是针对他一个人的。在他眼里，这个联盟的核心是林夜的战斗力、星辰宗的传承、星陨圣殿试炼胜利者的头衔。

但林夜身边的人呢？

铁蛋、莫言、苏琳——他们难道不是这个联盟的一部分吗？

星月沉默了很久。

"你说得对，"他终于开口，"我的想法太狭隘了。如果你们愿意加入，这个联盟的大门对你们所有人都是开放的。"

苏琳的眼神这才缓和了一些。

"我希望你能记住今晚说的话，"她说，"在这里，没有人是不可替代的。但也没有人是被排斥在外的。"

星月郑重地点了点头。

林夜看着他，心中却在思考着更深的问题。

这个联盟能够成功吗？星辰会内部的分裂是否会波及到他们？沈渊这个人——如果他真的与暗域有勾结，是否会派人来破坏？

太多未知数了。

但有一件事是确定的。

这场战争，已经不再是他一个人的战斗了。

**本章完**
