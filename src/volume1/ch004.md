# 第四章 废土狩猎

午后的废土，天色昏黄得像一具腐烂的尸体。

林夜蹲在一堵残破的矮墙后，屏住呼吸。他的眼睛眯成一条细缝，透过墙体的裂隙向外窥视。

二十米外，一条铁背蜈蚣正在翻找一堆垃圾。这条蜈蚣比林夜之前遇到的任何变异兽都要大——成人手臂粗细，深褐色的甲壳在阳光下泛着病态的光泽。它那数不清的节肢在碎石上发出细碎的沙沙声，听得人头皮发麻。

这不是他今天唯一的收获。

三天了。他已经在这片废墟区边缘游荡了三天，只猎到两只皮包骨头的变异鼠和一条剧毒蛇。废土上的猎物越来越少，而他的存粮已经见底。

如果今晚再没有收获，他就要饿着肚子过夜了。

而他腰间那把短刀的木鞘里，父亲留下的生锈短刀已经两天没开过荤了。

林夜的手无意识地触碰了一下那把短刀。刀柄上有一道深深的豁口，是父亲某次狩猎时留下的——或者说，是父亲最后一次狩猎时留下的。

那一年，林夜七岁。

他还记得那天早晨。父亲把他叫到跟前，将那把短刀交到他手里。"阿夜，"父亲的声音低沉而沙哑，"这把刀跟了爸十二年。今天开始，它就是你的了。"

那是他最后一次拥有父亲的声音。

七岁那年冬天，暗域生物袭击了他们的聚落。林夜躲在母亲身后，亲眼看着父亲举起那把短刀冲向三头黑影。母亲用身体护住他，用那把破旧的狩猎弓射光了最后一支箭。

然后母亲也倒下了。

林夜永远记得那一幕——父亲握着短刀的手，母亲背上的箭伤，还有那股从他心底深处涌起的、灼热得几乎要将他烧成灰烬的恨意。

他躲在母亲的尸体下，整整一天一夜。暗域生物的脚步声在他头顶走过，腐烂的气息渗透进他的衣服。他咬着牙，不发出一点声音。

从那以后，这把生锈的短刀和那张再也射不出箭的破弓，就成了他唯一的遗产。

"沙沙沙——"

蜈蚣的移动声将林夜从回忆中拉回。他甩了甩头，将那些画面强行压进脑海深处。现在不是想这些的时候。

他缓缓从腰间抽出短刀。

刀刃早已不复当年的锋利。三年的使用让刃口出现了细小的缺口，刀身上布满了擦不掉的锈迹。但如果他想今晚有地方睡、有东西吃，他就必须冒险。

林夜从矮墙后站起身。

铁背蜈蚣的触角立刻颤动起来——它感知到了他的气息。但它没有立刻逃跑，而是将那扁平的脑袋转向他的方向，像是在评估这个新出现的威胁。

林夜慢慢绕着蜈蚣移动，尽量让自己的脚步声轻一些。他在寻找一个机会——一个能让这只怪物分心的机会。

蜈蚣也在观察他。它的毒颚微微张开，露出那根正在颤动的毒刺，发出一种低沉的嘶嘶声。那是警告。

林夜没有停下脚步。他的眼睛盯着蜈蚣的身体，心中默默计算着距离和角度。

十米。

八米。

五米。

就在他即将进入攻击距离的瞬间，蜈蚣突然动了。

它的速度快得惊人——比林夜预想的还要快。那庞大的身躯猛然弹起，毒刺直刺他的胸口。

林夜的身体本能地向左侧扑倒，在地上翻滚了一圈。毒刺擦着他的肩膀刺过，毒液溅在身后的水泥地上，发出嗤嗤的腐蚀声，冒起一缕白烟。

好险。

但他没有时间庆幸。蜈蚣的第二击已经到来——这一次是横扫，巨大的尾节像一根钢鞭一样抽向他的腰部。

林夜来不及躲避。他只能举起左臂格挡。

"砰！"

剧烈的撞击让他的身体飞了出去，重重摔在五米外的碎石堆里。他的左臂传来一阵剧痛——不是毒液腐蚀的灼烧感，而是骨头裂开的声音。

断了一根骨头。

林夜咬紧牙关，从碎石堆里爬起来。左手软绵绵地垂在身侧，但他没有时间恐惧。蜈蚣已经再次逼近，那扭曲的身躯在废墟间穿行，如同一条活过来的噩梦。

他的右手紧紧握着短刀。

父亲用这把刀杀过比他更大的猎物。他告诉自己。这把刀还能用。它还能杀人。

蜈蚣的头部高高昂起，毒刺对准了他的眼睛。这一次，它打算一击毙命。

就在这千钧一发之际，林夜做出了一个疯狂的决定——他没有后退，而是迎着蜈蚣冲了上去。

蜈蚣的毒刺刺下的瞬间，他的身体突然下蹲，从蜈蚣的腹部下方滑过。与此同时，他手中的短刀狠狠向上刺去。

"噗——"

刀刃刺入蜈蚣柔软的腹部，甲壳在那个位置没有完全覆盖。但那把锈迹斑斑的短刀只刺入了一寸就再也进不去了——刀刃太钝，角度也太差。

蜈蚣发出震耳欲聋的惨叫。它的身躯剧烈翻滚，将林夜甩了出去。但林夜没有松手——他借着翻滚的力道，在蜈蚣的腹部划出一道长长的伤口。

黑色的血从伤口中涌出，溅了林夜一身。

但蜈蚣没有倒下。它疯狂地扭动着身躯，向四周无差别地攻击。节肢乱舞，毒刺横扫，将周围的一切都卷入混乱之中。

林夜被一条节肢抽中后背，整个人飞了出去，撞在一堵断墙上。背脊传来剧烈的疼痛，但他顾不上这些。他挣扎着爬起来，再次冲向蜈蚣。

不能停。一旦停下来，这只怪物就会逃进废墟深处。而他今天所有的努力都会白费。

他的左手已经废了，只能用右手握着那把短刀。刀刃上沾满了蜈蚣的血，变得滑腻腻的，几乎握不稳。

蜈蚣也受伤了。它的腹部有一道一尺长的伤口，黑色的血像泉水一样涌出。但它仍然在战斗——不，它在拼命。因为它也受伤了，而且伤得很重。

他们都在赌。

赌谁先倒下。

蜈蚣的头部再次对准林夜，那复眼里映出他浑身是血的身影。它的毒刺高高举起，准备发起最后的攻击。

林夜没有躲。

他的眼睛死死盯着蜈蚣的头部，心中突然闪过一个画面——

父亲举着短刀冲向三头黑影的背影。

母亲背上的箭伤。

还有那个躲在尸体下、浑身发抖却不敢发出一点声音的七岁男孩。

"你们杀了我父母……"

林夜的声音沙哑得不像他自己的。那是一种从地狱深处传来的声音，冰冷、绝望、而又充满了疯狂的恨意。

"……那我就杀了你们所有的同类。"

他冲了上去。

不是迎向蜈蚣的毒刺，而是迎向它的头部。他的身体压得很低，短刀在手中翻转，刃口朝上。

蜈蚣的毒刺刺下的瞬间，林夜的身体从它的下方穿过。毒刺擦过他的头发，刺入了地面。

而他的短刀，则刺入了蜈蚣的眼睛。

"嘶嘶嘶嘶——！！！"

蜈蚣发出撕心裂肺的惨叫。它的头疯狂地甩动，试图将林夜甩下来。但林夜已经顾不上那么多了。他的左手搂住蜈蚣的头部，右手握着短刀，一下、两下、三下——

不停地刺入那柔软的眼球。

一下、两下、三下。

直到黑色的血涌满了他的手掌。

直到蜈蚣的惨叫声渐渐微弱。

直到那庞大的身躯终于轰然倒地，再也不动。

林夜跪在蜈蚣的尸体旁，大口大口地喘着气。他的全身都是血，有蜈蚣的，也有他自己的。他的左臂以一种不自然的角度垂着，断骨已经刺穿了皮肤，露在外面。

疼。

但他感受不到疼。

他只感受到一种冰冷的平静。

"阿夜，这把刀跟了爸十二年。"

父亲的声音再次在他耳边响起。

"爸用它杀过比你更大的猎物。"

他低头看着手中的短刀。刀刃上全是缺口，刀身锈迹斑斑，但刀柄上那道深深的豁口依然清晰可见。

这是父亲的刀。

他用父亲的刀，杀死了第一只变异兽。

林夜慢慢将短刀收回鞘中，然后挣扎着站起来。他的双腿在发抖，每走一步都要用尽全身的力气。但他不能停下来。

蜈蚣的肉可以吃。虽然有毒，但烤熟之后能降低毒性。不多，但够他撑过今晚。

他用还能动的右手，从腰间摸出母亲留下的狩猎弓。弓身已经裂开了好几道口子，弓弦也只剩下最后一丝纤维，随时都可能断裂。

母亲就是用这张弓，射光了最后一支箭。

林夜将弓放回腰间，然后开始处理蜈蚣的尸体。他割下一块肉，用布包好，塞进怀里。然后又割了一块，大口大口地生吃起来。

恶心。

但他必须吃。他需要能量来修复伤口，需要力气来走回聚落。

生肉的口感像腐烂的泥巴，咽下去的时候他差点呕出来。但他强迫自己咽下去，一口接一口，直到胃里装不下为止。

然后他开始往回走。

走了大约一个小时，天色渐渐暗了下来。林夜找到一处相对完整的废弃建筑，钻了进去。

他的伤口已经开始发黑。毒素在他体内蔓延，每一次心跳都伴随着隐隐的刺痛。

但他没有解毒剂。他也没有钱买解毒剂。在废土上，药品比食物更珍贵，而他已经三天没有狩猎到足够的猎物来换取任何东西了。

林夜靠着墙壁坐下，将断掉的左臂用布条固定住。然后他从怀里摸出那块蜈蚣肉，放在火堆上烤。

火焰映在他苍白的脸上，勾勒出一道道阴影。

他的脑海中不断闪回那些画面——父亲冲向黑影的背影，母亲倒下的瞬间，还有那股他永远都无法忘记的、灼热的恨意。

"除了复仇……"

他的声音在空旷的废墟中回荡，低沉而沙哑。

"……我什么都不信。"

火光在他眼中跳动，映出一片冰冷的黑暗。
