# 第一百九十章 彼岸

黑暗在宇宙的深处翻涌。

那是一种难以名状的感觉——仿佛有什么东西正在苏醒，在虚无的深渊中缓缓睁开眼睛。林夜站在星际战舰的观景舷窗前，凝视着前方那片漆黑的虚空。他的眉头紧锁，心中涌起一股难以言喻的压迫感。

"你感觉到了吗？"苏琳的声音从他身后传来。

林夜点了点头。"虚无之渊……它在躁动。"

自从击败暗皇之后，虚无之渊的反噬就一直在加剧。那股曾经吞噬了整个星系的黑暗力量正在重新凝聚，在宇宙的边缘地带蠢蠢欲动。联盟的探测器已经多次捕捉到异常的能量波动，所有的数据都指向同一个事实——

大灾变正在逼近。

"宇宙意识传来的讯息，"苏琳走到他身旁，银色的长发在舷窗的微光中泛着淡淡的光泽，"每一万亿年一次的轮回劫难……而现在，距离下一次大灾变只剩不到五百年。"

五百年。

这个数字在林夜的心中沉甸甸的。对于凡人而言，五百年是难以企及的时间跨度；但对于修炼者而言，尤其是对于已经达到脉冲境巅峰的他来说，五百年不过是宇宙长河中的一瞬。

"我们必须做点什么。"林夜的声音低沉而坚定。

苏琳点了点头，她的预言之眼微微闪烁，仿佛在洞察着某个遥远的未来。

"有一个方法。"她说，"传说中，宇宙诞生之初曾诞生过五大奇点——时间、空间、引力、电磁、热力。这五大奇点是宇宙的基本法则的具象化，蕴含着能够重塑现实的力量。如果能够收集它们，或许能够找到对抗大灾变的方法。"

林夜转过身，直视着苏琳的眼睛。

"五大宇宙奇点……"

"是的。"苏琳的声音带着几分凝重，"但寻找它们的道路不会平坦。这些奇点分散在宇宙的各个角落，隐藏在连最强大的修炼者都难以企及的禁地之中。而且……"

她顿了顿，眼中闪过一丝忧虑。

"根据宇宙意识的指引，在我们收集奇点的过程中，将会遇到一位古老的存在——太初。他是宇宙诞生之初就已经存在的生灵，知道很多我们不知道的秘密。"

"太初……"林夜默默地将这个名字记在心中。

窗外的星光依然闪烁，但在那光芒的尽头，黑暗正在缓缓蠕动。虚无之渊的反噬越来越强，大灾变的阴影越来越近。林夜知道，他们即将踏上一段充满未知的旅程。

但他不会退缩。

因为在这个宇宙中，还有太多他想要守护的人。

"我们走吧。"林夜说，"前往第一个奇点所在的地方。"

苏琳点了点头，眼中闪烁着坚定的光芒。

"无论去哪里，我都陪着你。"

星际战舰缓缓启动，向着宇宙的深处驶去。而在他们的前方，是一段注定充满艰险的旅途——五大宇宙奇点的寻找，与古老存在太初的相遇，以及即将到来的五百年大灾变的威胁。

一切，才刚刚开始。
