# 第二百七十三章 引力的本质

当林夜和苏琳穿过最后一片虚空领域，那股无形的拉扯感终于近在眼前。

它不像之前那些星域中的引力场那样可以抵御，也不像黑洞的引力那样狂暴。它只是静静地存在于那里，散发着一种无法言喻的压迫感。

但当他们靠近时，才真正感受到它的可怕。

那不是普通的引力。

那是宇宙中最原始的力量。

所有的空间都在向它坍缩，所有的物质都在被它撕裂。无论是星辰、光芒，还是时间本身，都在被那股无形的力量拉向同一个方向——一个无法逃离的深渊。

"这是……"苏琳的声音有些颤抖。

"引力奇点的核心。"林夜轻声说，声音中带着难以抑制的震撼。

他终于明白了。

这片虚空并非普通的空间。这里的引力已经强大到超越了物理定律的范畴，所有的物质、能量、甚至是空间本身，都被压缩到了一个极致微小的点上。而那个点，就是引力奇点——宇宙中最强大的陷阱。

当他们的脚步踏入那片虚空的瞬间，整个空间都扭曲起来。

林夜感受到了前所未有的拉扯。不是来自某个方向的力，而是来自所有方向的力——上下、左右、前后、内外的界限已经模糊，一切都被那个微小的点所吸引。

他的身体在颤抖。

不是因为恐惧，而是因为身体在对抗那股无孔不入的引力。每一个细胞都在被拉扯，每一个分子都在被撕裂。他想要迈出一步，却发现自己已经无法分辨哪是前进，哪是后退。

引力的方向，就是一切的方向。

"林夜！"苏琳的声音从身侧传来。她也在承受着同样的拉扯，却依然在向他靠近。

他们的手终于握在一起。

那一刻，奇异的事情发生了。当两人的手相触的瞬间，那股强大的引力似乎减轻了一些。不是消失，只是分流——那股原本只作用于林夜一个人的引力，现在分摊到了两个人身上。

"连接……"林夜低声呢喃。

他想起了空间奇点时的领悟。连接是力量的来源。当两个人连接在一起，他们就能够分担原本无法承受的重量。

但引力奇点教给他的，还不止这些。

他看到了那些被拉向奇点的星辰。它们曾经闪耀过，有过自己的光辉，有过自己的生命。但现在，它们只是被引力压缩成了最基本的粒子，失去了所有的光芒和形态。

这就是引力的可怕之处。

它不是摧毁，而是抹杀。将所有的存在压缩到最原始的状态，让一切失去意义。

"暗皇……"林夜低声呢喃。

他突然明白了。

暗皇的孤独，就是一种引力。

一种将自己拉向毁灭的引力。
