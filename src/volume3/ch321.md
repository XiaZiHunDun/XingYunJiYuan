# 第三百二十一章 决战前夜

篝火在夜色中静静燃烧，火光将几道身影拉得很长很长。

那是决战前夜的最后一刻。星光在头顶流转，宇宙的深处传来阵阵低沉的轰鸣，那是虚无之主正在苏醒的征兆。明日之后，一切都将改变——或者归于永恒的黑暗，或者迎来光明与黑暗的重新融合。

林夜坐在一块岩石上，火光在他脸上跳跃。他的目光越过铁蛋和莫言，落在远方那片看不见的黑暗上。

"铁蛋，莫言。"他开口，声音低沉而平静，"我有话要和你们说。"

铁蛋和莫言对视一眼。他们从林夜的语气中听出了什么——那不是命令，不是商量，而是一种托付。

"老大？"铁蛋向前走了一步，眉头微微皱起。

林夜没有立刻回答。他从岩石上站起来，火光在他身后形成一道长长的影子。他的目光落在铁蛋身上，那目光中有一种铁蛋从未见过的东西——不是审视，不是期望，而是一种深沉的、带着温度的信任。

"铁蛋，你跟我多少年了？"林夜问。

铁蛋愣了一下，然后认真地计算起来。那些岁月太过漫长，漫长到已经成为了生命的一部分。

"从废土开始，"他说，"一千七百三年。"

林夜点了点头。一千七百三年。那是一段漫长的岁月，漫长到足以让一个废土上的孤儿成长为可以独当一面的守护者。

"一千七百三年，"林夜重复了一遍，声音中有一丝不易察觉的波动，"你是我带出来的人中，跟我最久的一个。"

铁蛋没有说话。他能感觉到林夜话语中的重量，那种重量让他的心脏微微收紧。

林夜向前走了一步。他抬起手，轻轻地落在铁蛋的肩膀上。那动作很轻，轻得像是拍去一片落叶，但其中包含的情感却重如泰山。

"如果我回不来，"林夜说，声音平静得像是陈述一个事实，"玄机阁就交给你了。"

篝火噼啪作响。

铁蛋的眼眶在一瞬间红了。那红来得太快，快到他来不及掩饰。他张了张嘴，想说什么，却发现喉咙像是被什么东西堵住了。

玄机阁。那是林夜一手建立的地方，是无数岁月的结晶，是这个宇宙中无数人仰望的圣地。而现在，林夜把这份重担托付给了他。

铁蛋深深地吸了一口气。他想说自己一定能做到，想说林夜一定会回来，想说很多很多话。但最后，他只是重重地点了点头。

"老大。"他的声音有些沙哑，"我明白了。"

林夜看着他的眼睛。那双眼睛里有泪光，但没有流下来。铁蛋已经不是当年那个冲动的年轻人了，他学会了把情感藏在心底，学会了用沉默来表达最深的承诺。

林夜的手在铁蛋的肩膀上停留了一瞬。然后，他轻轻地拍了拍他。

没有多说。

有些话不需要多说。一千七百三年的岁月已经说明了一切。

铁蛋抬起手，用力地揉了揉眼睛。他不想让林夜看到他的脆弱。但当他放下手的时候，他的目光已经恢复了清明。

"老大，"他说，声音已经完全恢复了平静，"你一定会回来的。"

这不是疑问，而是陈述。是一种不容置疑的信念。

林夜微微一笑。

"我也这么希望。"他说。

然后他的目光转向莫言。

"莫言，"他说，"铁蛋就交给你了。"

莫言微微一怔，然后郑重地点了点头。她知道这句话的重量。一千七百三年，林夜把铁蛋交给她，不只是因为信任，而是因为责任。

"我会的。"她说，声音轻但坚定。

铁蛋看了她一眼，眼中有什么在闪烁。

篝火继续燃烧。夜风从远处吹来，带着宇宙深处的气息。明日，一切都将改变。但此刻，在篝火旁，他们只是三个即将分别的战友。

林夜转过身，看向远方的黑暗。

"去休息吧。"他说，"明天，将是漫长的一天。"

铁蛋和莫言没有动。他们看着林夜的背影，那背影在火光中显得孤独而坚定。

"老大。"铁蛋突然开口。

林夜回过头。

"我们在地上等你。"铁蛋说，"和所有人一起，等你回来。"

林夜没有说话。他只是微微点了点头，然后转回身，继续看向那片黑暗。

在那片黑暗的最深处，虚无之主正在等待。

而明天，他将带着苏琳，去面对那最终的挑战。

星光在头顶流转，见证着这个决战前夜的承诺。
